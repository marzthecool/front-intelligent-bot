### Installation

_To get started with the project._

1. Clone the repo
   ```sh
   git clone https://github.com/marzthecool/front-intelligent-bot.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. npm
  ```sh
  npm install npm@latest -g
  ```
4. Start the application
   ```sh
   npm start
   ```
